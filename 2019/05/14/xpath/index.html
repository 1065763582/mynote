XPath

Posted on 2019-05-14 | Edited on 2019-05-17 | Views:

XPath 是一门在 XML 文档中查找信息的语言，属于 W3C 标准，包含一个标准函数库。XPath 用于在 XML 文档中通过元素和属性进行导航。在 XPath 中，有七种类型的节点：

  1. 元素
  2. 属性
  3. 文本
  4. 命名空间
  5. 处理指令
  6. 注释
  7. 文档节点（或称为根节点）

节点间的关系：父节点 子节点 同胞节点 先辈节点 后代节点
